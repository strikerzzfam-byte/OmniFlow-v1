# OmniFlow - ArcNex Core

Next-gen all-in-one workspace that unifies writing, design, and content generation in a single futuristic interface.

## Getting Started

To run this project locally:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Features

- **OmniWrite**: Intelligent writing assistant with real-time suggestions
- **OmniDesign**: Drag-and-drop design lab with instant mockups
- **OmniGenerate**: AI-powered content generation

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
